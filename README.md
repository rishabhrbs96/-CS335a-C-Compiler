## How to run 
```sh
$ cd src
$ make 
$ cd ..
```
There are 5 code files name
* code1.c
* code2.c
* code3.c
* code4.c
* code5.c
```sh
$ ./bin/parser.sh ./test/code1.c
```
The parse tree is generated in the "test" folder.
```sh
$ cd test
```

After the execution go to the src directory and run.
```sh
$ make clean
```
